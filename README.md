## vite-vue3-react-svelte-todo

one todo-app with vite + vue3 + react + svelte.

### preview

vue: https://vite-vue3-todo.netlify.app

react: https://vite-react-todo.netlify.app

### quick Start 
```
cd vite-vue3-react-svelte-todo && yarn 

```

vue3:
```
yarn vue3-dev
```

react:
```
yarn react-dev
```

svelte:
```
yarn svelte-dev
```
